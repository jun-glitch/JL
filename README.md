# Birder
25-2 캡스톤디자인과창업프로젝트A Team10 JL
***
#### 프로젝트명
탐조 입문자를 위한 Image Classification 기반 실시간 새 종 판별 및 새 관측 기록 제공 서비스<br>
#### 주요 기능 <br> 
1. 위치 정보를 반영한 이미지 검색<br>
2. 위치 별, 종 별 실시간 관측 기록 검색<br>
3. 개인 디지털 도감 제공<br>
***
#### Tech Stack

##### Backend
- Python 3.10
- Django 5.x
- Django REST Framework
- PostgreSQL + PostGIS

##### Frontend
- Flutter

##### AI / Machine Learning
- PyTorch

##### Open API
- Wikimedia Commons API (학습 이미지 수집)
- eBird API (종 정보 확보)
